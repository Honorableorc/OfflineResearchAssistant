## вљ™ Installation

### Clone Repository

```bash
git clone https://github.com/your-username/OfflineResearchAssistant.git

cd OfflineResearchAssistant
```

### Create Virtual Environment

```bash
python -m venv .venv
```

### Activate Environment

Windows:

```bash

```

Linux/Mac:

```bash
source .venv/bin/activate
```

### Install Dependencies

